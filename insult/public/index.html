RHOAR Shootout - Node.js

This application showcases a silly implementation of an application built using the microservices design pattern. There are three services involved, in addition to the front end (which you are currently viewing). Those services are 

GET http://adjective:8080/api/adjective // provides an adjective
GET http://noun:8080/api/noun // provides a noun
GET http://insult:8080/api/insult // provides an insult

Click the big blue button to be insulted.
